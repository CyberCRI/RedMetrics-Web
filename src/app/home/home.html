Welcome to RedMetrics

RedMetrics offers open source game analytics. It is made up of a RESTful web service and a web app that allows game designers, researchers, and teachers to track game metrics, then download the raw data for offline analysis.

Like existing services, RedMetrics allows you to track player progression through your game. But RedMetrics goes a step further by also tracking snapshots - a recording of gameplay that can include game state such as entities, player input, and anything else that a designers wish to track.

In the spirit of open data, all data gathered on RedMetrics is freely available, with players only identified by anonymous identifiers. Data can be viewed via the website or downloaded for statistical analysis.

RedMetrics has been designed to integrate nicely with RedWire, an open source game engine for remixing and mashups. But as a web service, any internet-connected game can use it.

To use RedMetrics for your game, check out the web api. You can use the hosted version at redmetrics.io or download the software and run your own instance.
